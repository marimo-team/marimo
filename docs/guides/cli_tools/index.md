# marimo tools

The `marimo tools` command contains utilities for working with notebooks.

| Tool | Description |
| :--- | :---------- |
| [Thumbnails](thumbnails.md) | Generate notebook thumbnail images. |

Thumbnails are used by [OpenGraph previews](../publishing/opengraph.md) and [gallery mode](../apps.md#gallery-mode).

For a full reference of all subcommands and flags, see [Commands](../../cli.md#marimo-tools).
