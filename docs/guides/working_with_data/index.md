# Working with data

These guides introduce you to marimo's features for working with data,
including SQL cells, no-code dataframe transformation tools, plots whose
selections are automatically sent back to Python, and browsing remote storage.

| Guide | Description |
|-------|-------------|
| [SQL](sql.md) | Use SQL to query dataframes, databases, CSVs, etc. |
| [Dataframes](dataframes.md) | Filter, search, and transform dataframes without code |
| [Plotting](plotting.md) | Send plot selections to Python |
| [Remote Storage](remote_storage.md) | Browse and download files from S3, GCS, Azure, and more |
